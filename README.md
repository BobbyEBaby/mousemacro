Download MouseKeyboardMacro.zip below, extract, and run the exe. No installation needed.

Record mouse movement and clicks, then replay them
Loop playback endlessly or a set number of times, with live loop countdown
Global hotkey to trigger playback from any app (instant, no countdown)
Configurable stop keys for recording and playback
Single self-contained exe - nothing else to install
Note: Windows SmartScreen will warn on first run because the exe is unsigned. Click More info -> Run anyway.

Requires 64-bit Windows 10/11 with a Vulkan-capable GPU.
